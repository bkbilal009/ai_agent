# ai_agent
This repository contains a beginner level Agentic AI project that fetches real world time for different cities using Python.


 Time Telling Agentic AI

This project is a beginner-level Agentic AI built using Python and Google Colab.  
It answers questions like current time in different cities using a tool function.

 Features
- Fetches real world time for different cities
- Interactive agent logic that understands user queries
- Runs directly in Google Colab

 How to Run
1. Open the notebook `Agentic_AI_Time_Agent.ipynb` in Google Colab
2. Run all cells
3. Ask time related questions like "What time in Lahore?"
